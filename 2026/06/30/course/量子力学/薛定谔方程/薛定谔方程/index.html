波函数的命运由谁决定？——薛定谔方程：微观世界的运动定律

上一篇 《一个电子穿过双缝，会留下什么？》 我们弄明白了一件事：微观粒子的状态要用一个波函数 Ψ(r,t)\Psi(\mathbf{r},t)Ψ(r,t) 来描写，它的模平方 ∣Ψ∣2|\Psi|^2∣Ψ∣2 给出粒子在空间各处出现的概率密度。

但文章结尾，留下了一个更要命的问题：

这个波函数，是怎么随时间变化的？ 给定了某个时刻的 Ψ\PsiΨ，下一秒它会变成什么样？是什么规律，在"指挥"概率分布的演化？

经典力学里，这个问题牛顿早就回答了——给定位置和速度，牛顿方程 F=maF=maF=ma 就能算出粒子此后每一刻的去向。可量子世界没有确定的轨道，F=maF=maF=ma 失灵了。我们需要一条全新的"运动定律"。

这条定律，就是大名鼎鼎的薛定谔方程（Schrödinger equation）。

iℏ ∂∂t Ψ(r,t)=H^ Ψ(r,t)i\hbar\,\dfrac{\partial}{\partial t}\,\Psi(\mathbf{r},t) = \hat{H}\,\Psi(\mathbf{r},t) iℏ∂t∂​Ψ(r,t)=H^Ψ(r,t)

别被它吓到。这一篇，我们就来一步步看清：这个方程是怎么"想"出来的，它为什么长这样，以及它凭什么能统辖整个微观世界。

一、薛定谔与他的方程：量子力学的"牛顿定律"

故事要从 1926 年说起。那时海森堡的矩阵力学刚刚诞生几个月，一位奥地利物理学家薛定谔（Erwin Schrödinger，1887—1961）受德布罗意物质波的启发，走上了另一条路：既然电子是波，那它就该满足一个波动方程。

他在 1926 年初接连发表六篇论文，创立了波动力学，写出了那个以他名字命名的方程。很快人们发现，薛定谔的波动力学和海森堡的矩阵力学其实是等价的——同一条规律的两种讲法。凭借这项工作，薛定谔和狄拉克共享了 1933 年诺贝尔物理学奖（“原子理论的新形式”）。

要理解这个方程的地位，记住三句话就够了：

  • 它描述的是一个质量为 μ\muμ 的非相对论实物粒子，在势场 U(r)U(\mathbf{r})U(r) 中状态随时间的变化——反映了微观粒子的运动规律。
  • 它是量子力学最基本的方程，地位相当于经典力学里的牛顿方程。量子力学对粒子运动的研究，基本上归结为在各种条件下求解薛定谔方程。
  • 它不是"推导"出来的，而是"建立"的——依据实验事实和基本假定提出，对不对要由实验来检验。

最后一点很关键，也常让初学者困惑：这么重要的方程，竟然不是从更基本的原理"证明"出来的？

这其实是物理学的常态。牛顿方程 F=maF=maF=ma 也不是推导出来的，它是牛顿基于大量观测"归纳"出来的基本定律。薛定谔方程同理——它是量子力学的基本假定，是逻辑链条的起点而非终点。它的"正确性"，靠的是一百多年来无数次实验的严格检验。

那么，薛定谔当年是怎么"建立"这个方程的呢？

二、方程是怎么"凑"出来的：从自由粒子出发

薛定谔方程虽然不能被"证明"，但它也不是凭空捏造的。它的建立遵循几条非常自然的"设计要求"：

[设计要求]

  1. 方程应该是关于波函数对时间微商的微分方程——因为我们想算的是 Ψ\PsiΨ 怎么随 ttt 变化；
  2. 方程必须是线性的（这是态叠加原理的要求：Ψ1\Psi_1Ψ1​ 和 Ψ2\Psi_2Ψ2​ 是解，它们的叠加也必须是解）；
  3. 方程的系数不应包含状态的参量（动量、能量这些"因状态而异"的量，不能写死在方程里）。

出发点，是我们已经知道的一个波函数——自由粒子的德布罗意波。一个动量为 p\mathbf{p}p、能量为 EEE 的自由粒子，其波函数是一个平面波：

Ψ(r,t)=A ei(p⋅r−Et)/ℏ\Psi(\mathbf{r},t) = A\,e^{i(\mathbf{p}\cdot\mathbf{r}-Et)/\hbar} Ψ(r,t)=Aei(p⋅r−Et)/ℏ

让我们对这个波函数分别"动一下手"——对时间求偏导，对空间求偏导（拉普拉斯算符 ∇2\nabla^2∇2）。

先对时间 ttt 求偏导：

iℏ ∂Ψ∂t=E Ψi\hbar\,\dfrac{\partial \Psi}{\partial t} = E\,\Psi iℏ∂t∂Ψ​=EΨ

再对空间求二阶偏导（用 ∇2=∂2∂x2+∂2∂y2+∂2∂z2\nabla^2 = \dfrac{\partial^2}{\partial x^2}+\dfrac{\partial^2}{\partial y^2}+\dfrac{\partial^2}{\partial z^2}∇2=∂x2∂2​+∂y2∂2​+∂z2∂2​）：

−ℏ2 ∇2Ψ=p2 Ψ-\hbar^2\,\nabla^2\Psi = p^2\,\Psi −ℏ2∇2Ψ=p2Ψ

而自由粒子的能量—动量关系是 E=p22μE = \dfrac{p^2}{2\mu}E=2μp2​。把上面两式一结合，消去 EEE 和 p2p^2p2，就得到自由粒子的薛定谔方程：

iℏ ∂Ψ∂t=−ℏ22μ ∇2Ψi\hbar\,\dfrac{\partial \Psi}{\partial t} = -\dfrac{\hbar^2}{2\mu}\,\nabla^2\Psi iℏ∂t∂Ψ​=−2μℏ2​∇2Ψ

漂亮！这个方程里已经不含 EEE、ppp 这些随状态改变的参量了，只剩下常数 ℏ\hbarℏ、μ\muμ 和对 Ψ\PsiΨ 的运算——正好满足设计要求。

更妙的是，对照上面两个求导结果，可以发现一个深刻的对应关系：

  E  ⟶  iℏ ∂∂t,p  ⟶  −iℏ ∇  \boxed{\;E \;\longrightarrow\; i\hbar\,\dfrac{\partial}{\partial t}, \qquad \mathbf{p} \;\longrightarrow\; -i\hbar\,\nabla\;} E⟶iℏ∂t∂​,p⟶−iℏ∇​

也就是说：把能量 EEE 换成算符 iℏ∂∂ti\hbar\dfrac{\partial}{\partial t}iℏ∂t∂​，把动量 p\mathbf{p}p 换成算符 −iℏ∇-i\hbar\nabla−iℏ∇，自由粒子的能量关系 E=p22μE=\dfrac{p^2}{2\mu}E=2μp2​ 就自动"翻译"成了薛定谔方程。这正是上一篇讲过的——力学量在量子力学里变成了算符。

🧠 小测一下：自由粒子的能量关系 E=p2/2μE=p^2/2\muE=p2/2μ 里，如果把 EEE 和 p\mathbf{p}p 都替换成对应的算符，会得到什么？

答案正是上面的自由粒子薛定谔方程 iℏ∂Ψ∂t=−ℏ22μ∇2Ψi\hbar\dfrac{\partial\Psi}{\partial t}=-\dfrac{\hbar^2}{2\mu}\nabla^2\Psiiℏ∂t∂Ψ​=−2μℏ2​∇2Ψ。算符替换，是建立薛定谔方程的那把"钥匙"。

三、加上势场，方程完整了

自由粒子太理想化了。现实中的粒子总在某种势场 U(r)U(\mathbf{r})U(r) 里运动（比如氢原子里的电子处在核的库仑势中）。这时粒子的总能量是动能加势能：

E=p22μ+U(r)E = \dfrac{p^2}{2\mu} + U(\mathbf{r}) E=2μp2​+U(r)

套用同样的算符替换，立刻得到单粒子的薛定谔方程（也称薛定谔波动方程）：

  iℏ ∂∂t Ψ(r,t)=[−ℏ22μ∇2+U(r)]Ψ(r,t)  \boxed{\;i\hbar\,\dfrac{\partial}{\partial t}\,\Psi(\mathbf{r},t) = \left[-\dfrac{\hbar^2}{2\mu}\nabla^2 + U(\mathbf{r})\right]\Psi(\mathbf{r},t)\;} iℏ∂t∂​Ψ(r,t)=[−2μℏ2​∇2+U(r)]Ψ(r,t)​

这就是薛定谔方程最常见的模样。等号右边方括号里的那一大坨，对波函数进行操作的"运算整体"，有一个专门的名字——哈密顿算符（Hamiltonian） H^\hat{H}H^：

H^=−ℏ22μ∇2+U(r)\hat{H} = -\dfrac{\hbar^2}{2\mu}\nabla^2 + U(\mathbf{r}) H^=−2μℏ2​∇2+U(r)

于是薛定谔方程可以写成最简洁、也最普遍的形式：

iℏ ∂Ψ∂t=H^ Ψi\hbar\,\dfrac{\partial \Psi}{\partial t} = \hat{H}\,\Psi iℏ∂t∂Ψ​=H^Ψ

第一项 −ℏ22μ∇2Ψ-\dfrac{\hbar^2}{2\mu}\nabla^2\Psi−2μℏ2​∇2Ψ 对应动能 p22μ\dfrac{p^2}{2\mu}2μp2​（动量算符作用两次），第二项 U(r)ΨU(\mathbf{r})\PsiU(r)Ψ 就是势能。哈密顿算符，本质上就是"总能量"变成了算符。

对于多个粒子组成的体系，思路完全一样，只是波函数变成所有粒子坐标的函数 Ψ(r1,r2,…,rN;t)\Psi(\mathbf{r}_1,\mathbf{r}_2,\dots,\mathbf{r}_N;t)Ψ(r1​,r2​,…,rN​;t)，哈密顿算符里要加上每个粒子的外势场和粒子之间的相互作用：

H^=∑i=1N(−ℏ22μi∇i2+U(ri))+∑i<jNV(ri,rj)\hat{H} = \sum_{i=1}^{N}\left(-\dfrac{\hbar^2}{2\mu_i}\nabla_i^2 + U(\mathbf{r}_i)\right) + \sum_{i<j}^{N} V(\mathbf{r}_i,\mathbf{r}_j) H^=i=1∑N​(−2μi​ℏ2​∇i2​+U(ri​))+i<j∑N​V(ri​,rj​)

形式复杂了，但灵魂不变：动能 + 势能，统统变成算符。

  • 自由粒子：U=0U=0U=0，H^=−ℏ22μ∇2\hat{H}=-\dfrac{\hbar^2}{2\mu}\nabla^2H^=−2μℏ2​∇2；

列出哈密顿量，只是万里长征第一步——真正的硬仗，是把这个方程解出来。

四、概率守恒：粒子不会凭空消失

薛定谔方程里藏着一个非常漂亮的性质：粒子的概率是守恒的——它不会凭空产生，也不会凭空消失，只能从一个地方"流"到另一个地方。

我们来推导一下。定义粒子的概率密度（上一篇讲过）：

w(r,t)=∣Ψ(r,t)∣2=Ψ∗Ψw(\mathbf{r},t) = |\Psi(\mathbf{r},t)|^2 = \Psi^*\Psi w(r,t)=∣Ψ(r,t)∣2=Ψ∗Ψ

它随时间怎么变？对 ttt 求偏导，∂w∂t=Ψ∗∂Ψ∂t+∂Ψ∗∂tΨ\dfrac{\partial w}{\partial t}=\Psi^*\dfrac{\partial\Psi}{\partial t}+\dfrac{\partial\Psi^*}{\partial t}\Psi∂t∂w​=Ψ∗∂t∂Ψ​+∂t∂Ψ∗​Ψ。把薛定谔方程（及其复共轭）代进去化简，可以得到：

∂w∂t+∇⋅J=0\dfrac{\partial w}{\partial t} + \nabla\cdot\mathbf{J} = 0 ∂t∂w​+∇⋅J=0

这就是鼎鼎大名的连续性方程。其中

J=ℏ2iμ(Ψ∗∇Ψ−Ψ∇Ψ∗)\mathbf{J} = \dfrac{\hbar}{2i\mu}\big(\Psi^*\nabla\Psi - \Psi\nabla\Psi^*\big) J=2iμℏ​(Ψ∗∇Ψ−Ψ∇Ψ∗)

叫作概率流密度矢量。

这两个量分别是什么意思？

  • w(r,t)w(\mathbf{r},t)w(r,t) 是 ttt 时刻在 r\mathbf{r}r 处单位体积内找到粒子的概率；
  • J(r,t)\mathbf{J}(\mathbf{r},t)J(r,t) 则描写概率在空间流动的强度和方向——就像电流密度描写电荷的流动一样。

把连续性方程对一个体积 VVV 积分，并用高斯定理把体积分变成面积分，物理意义就一目了然了：

ddt∫V∣Ψ∣2 dτ=−∮SJ⋅dS\dfrac{d}{dt}\int_V |\Psi|^2\,d\tau = -\oint_S \mathbf{J}\cdot d\mathbf{S} dtd​∫V​∣Ψ∣2dτ=−∮S​J⋅dS

左边是"体积 VVV 内的总概率随时间的增加率"；右边的负号意味着，它等于从外界穿过边界面 SSS 流进 VVV 内的概率通量。说白了：VVV 里面概率变多了，一定是从外面流进来的；变少了，一定是流出去了。 概率像流体一样守恒。

如果进一步假定这个量子系统是局域的——在无穷远处既没有粒子注入也没有流出（不存在净粒子流），那么当 V→∞V\to\inftyV→∞ 时，穿过无穷远界面的通量为零，于是：

ddt∫∞∣Ψ(r,t)∣2 dτ=0\dfrac{d}{dt}\int_\infty |\Psi(\mathbf{r},t)|^2\,d\tau = 0 dtd​∫∞​∣Ψ(r,t)∣2dτ=0

也就是说，粒子在全空间出现的总概率不随时间改变。如果一开始归一化为 1，它就永远是 1——粒子总在某处，绝不会"蒸发"。

这正是上一篇"归一化"能成立的根本保证：薛定谔方程自动保证了总概率守恒，所以一旦归一化，就永远是归一化的。

薛定谔方程内蕴的这条性质，与势场 U(r)U(\mathbf{r})U(r) 的具体形式无关：就任意局部空间而言，粒子数定域守恒；就全空间而言，总粒子数守恒。 波函数的变化，仅仅表示粒子在时空中的运动。

这其实划定了非相对论量子力学的适用边界：它不考虑粒子的产生和湮灭，不考虑粒子之间的相互转化（比如电子-正电子对湮灭成光子）。要处理那些现象，得走到相对论量子力学和量子场论。这正是传统意义上"力学"的范畴——研究现有物体的运动，而非物体的生灭。

五、顺手得到：质量守恒与电荷守恒

概率守恒既然成立，把概率密度乘上粒子的质量 μ\muμ 或电荷 eee，就立刻得到质量守恒和电荷守恒：

守恒量 密度 流密度
概率 w=∣Ψ∣2w=|\Psi|^2w=∣Ψ∣2 J=ℏ2iμ(Ψ∗∇Ψ−Ψ∇Ψ∗)\mathbf{J}=\dfrac{\hbar}{2i\mu}(\Psi^*\nabla\Psi-\Psi\nabla\Psi^*)J=2iμℏ​(Ψ∗∇Ψ−Ψ∇Ψ∗)
质量 wμ=μw=μ∣Ψ∣2w_\mu=\mu w=\mu|\Psi|^2wμ​=μw=μ∣Ψ∣2 Jμ=μJ\mathbf{J}_\mu=\mu\mathbf{J}Jμ​=μJ
电荷 we=ew=e∣Ψ∣2w_e=ew=e|\Psi|^2we​=ew=e∣Ψ∣2 Je=eJ\mathbf{J}_e=e\mathbf{J}Je​=eJ

它们都满足同样形式的连续性方程 ∂ρ∂t+∇⋅Jρ=0\dfrac{\partial \rho}{\partial t}+\nabla\cdot\mathbf{J}_\rho=0∂t∂ρ​+∇⋅Jρ​=0。

也就是说，薛定谔方程不仅预言了概率守恒，还自动给出了质量守恒和电荷守恒——一条方程，三重守恒。这正是一个好物理理论的魅力：它蕴含的，远比你放进去的多。

六、定态薛定谔方程：把时间"分离"出去

解一个既含时间又含空间的偏微分方程，可不容易。但有一类极其重要的问题——势能 U(r)U(\mathbf{r})U(r) 不显含时间（稳定的势场）——可以让事情大大简化。氢原子中的电子（库仑势只依赖 rrr）、自由粒子（U=0U=0U=0）都属于此类。

这时，可以用分离变量法：设波函数是空间部分和时间部分的乘积

Ψ(r,t)=ψ(r) f(t)\Psi(\mathbf{r},t) = \psi(\mathbf{r})\,f(t) Ψ(r,t)=ψ(r)f(t)

代回薛定谔方程，时间部分和空间部分就可以分开。时间部分给出

f(t)=e−iEt/ℏf(t) = e^{-iEt/\hbar} f(t)=e−iEt/ℏ

而空间部分 ψ(r)\psi(\mathbf{r})ψ(r) 满足的方程，就是定态薛定谔方程（也叫不含时薛定谔方程，或势场中的能量本征方程）：

  H^ ψ(r)=[−ℏ22μ∇2+U(r)]ψ(r)=E ψ(r)  \boxed{\;\hat{H}\,\psi(\mathbf{r}) = \left[-\dfrac{\hbar^2}{2\mu}\nabla^2 + U(\mathbf{r})\right]\psi(\mathbf{r}) = E\,\psi(\mathbf{r})\;} H^ψ(r)=[−2μℏ2​∇2+U(r)]ψ(r)=Eψ(r)​

这里出现的常数 EEE，物理意义就是粒子的能量。相应的"定态波函数"为

Ψ(r,t)=ψ(r) e−iEt/ℏ\Psi(\mathbf{r},t) = \psi(\mathbf{r})\,e^{-iEt/\hbar} Ψ(r,t)=ψ(r)e−iEt/ℏ

注意区分两个方程：含时的 iℏ∂Ψ∂t=H^Ψi\hbar\dfrac{\partial\Psi}{\partial t}=\hat{H}\Psiiℏ∂t∂Ψ​=H^Ψ 描写状态如何演化；不含时的 H^ψ=Eψ\hat{H}\psi=E\psiH^ψ=Eψ 是一个本征值方程，要解决的是"在这个势场里，粒子允许有哪些能量、对应的波函数长什么样"。量子力学里大量的计算，本质上都是在解这个本征方程。

定态 vs 非定态

定态（stationary state）：体系的能量有确定值的状态（即能量本征态 H^ψ=Eψ\hat{H}\psi=E\psiH^ψ=Eψ 的解）。在定态中，体系的各种力学性质不随时间改变。

非定态：由若干个能量不同的本征态叠加而成的态。

处于定态时，粒子有三个鲜明特征：

  1. 概率密度 ∣Ψ∣2=∣ψ∣2|\Psi|^2 = |\psi|^2∣Ψ∣2=∣ψ∣2 和概率流密度 J\mathbf{J}J 都不随时间改变——这正是"定"字的含义；
  2. 任何不显含 ttt 的力学量，其平均值不随时间改变；
  3. 任何不显含 ttt 的力学量，其测值的概率分布也不随时间改变。

A. 定态是不随时间变化的量子态　B. 定态是能量的本征态　C. 定态是能量有确定值的状态　D. 由波函数 ΨE(r,t)=ψE(r)e−iEt/ℏ\Psi_E(\mathbf{r},t)=\psi_E(\mathbf{r})e^{-iEt/\hbar}ΨE​(r,t)=ψE​(r)e−iEt/ℏ 描写的状态是定态

答案是 A、B、C、D 全对。定态就是能量本征态，能量取确定值，且各种力学性质（概率密度、平均值、概率分布）都不随时间变化。

🧠 再测一个：处于定态时，粒子的概率密度和概率流密度随时间怎么变？

答案是 不变。因为 ∣Ψ∣2=∣ψ(r)e−iEt/ℏ∣2=∣ψ(r)∣2|\Psi|^2 = |\psi(\mathbf{r})e^{-iEt/\hbar}|^2 = |\psi(\mathbf{r})|^2∣Ψ∣2=∣ψ(r)e−iEt/ℏ∣2=∣ψ(r)∣2，时间相位 e−iEt/ℏe^{-iEt/\hbar}e−iEt/ℏ 一取模平方就消掉了。

七、叠加态：当时间重新"流动"

既然单个定态的概率分布是"死"的（不随时间变），那量子世界里那些随时间振荡的现象（比如电子在原子里的运动感、跃迁过程中的节奏）从何而来？

答案藏在态叠加原理里。薛定谔方程是线性方程，所以它的通解，是所有能量本征态的线性叠加：

Ψ(r,t)=∑ncn ψn(r) e−iEnt/ℏ\Psi(\mathbf{r},t) = \sum_{n} c_n\,\psi_n(\mathbf{r})\,e^{-iE_n t/\hbar} Ψ(r,t)=n∑​cn​ψn​(r)e−iEn​t/ℏ

其中 ψn\psi_nψn​ 是能量为 EnE_nEn​ 的本征态，系数 cnc_ncn​ 由初始状态决定（且满足归一化 ∑∣cn∣2=1\sum|c_n|^2=1∑∣cn​∣2=1）。

关键来了：叠加态不再是定态！ 这时概率密度

∣Ψ(r,t)∣2=∣∑ncnψn(r)e−iEnt/ℏ∣2|\Psi(\mathbf{r},t)|^2 = \left|\sum_n c_n\psi_n(\mathbf{r})e^{-iE_n t/\hbar}\right|^2 ∣Ψ(r,t)∣2=​n∑​cn​ψn​(r)e−iEn​t/ℏ​2

由于各项带有不同的时间相位 e−iEnt/ℏe^{-iE_n t/\hbar}e−iEn​t/ℏ，取模平方时会产生交叉项，于是 ∣Ψ∣2|\Psi|^2∣Ψ∣2 和 J\mathbf{J}J 都会随时间振荡——时间，重新开始"流动"了。

举个最简单的例子：假设一个粒子初始时刻是两个定态的等权叠加

Ψ(x,0)=c1ψ1(x)+c2ψ2(x)\Psi(x,0) = c_1\psi_1(x) + c_2\psi_2(x) Ψ(x,0)=c1​ψ1​(x)+c2​ψ2​(x)

为简单起见，设 c1,c2c_1,c_2c1​,c2​ 和 ψ1,ψ2\psi_1,\psi_2ψ1​,ψ2​ 都是实的。那么任意时刻

Ψ(x,t)=c1ψ1(x)e−iE1t/ℏ+c2ψ2(x)e−iE2t/ℏ\Psi(x,t) = c_1\psi_1(x)e^{-iE_1 t/\hbar} + c_2\psi_2(x)e^{-iE_2 t/\hbar} Ψ(x,t)=c1​ψ1​(x)e−iE1​t/ℏ+c2​ψ2​(x)e−iE2​t/ℏ

利用欧拉公式 e−iθ=cos⁡θ−isin⁡θe^{-i\theta}=\cos\theta-i\sin\thetae−iθ=cosθ−isinθ 算出 ∣Ψ∣2|\Psi|^2∣Ψ∣2，会发现（化简后）

∣Ψ(x,t)∣2=c12ψ12+c22ψ22+2c1c2 ψ1ψ2 cos⁡ ⁣(E2−E1ℏ t)|\Psi(x,t)|^2 = c_1^2\psi_1^2 + c_2^2\psi_2^2 + 2c_1c_2\,\psi_1\psi_2\,\cos\!\left(\dfrac{E_2-E_1}{\hbar}\,t\right) ∣Ψ(x,t)∣2=c12​ψ12​+c22​ψ22​+2c1​c2​ψ1​ψ2​cos(ℏE2​−E1​​t)

看最后一项：概率密度在空间每一点都以正弦（余弦）形式随时间振动，振动的角频率是

ω=E2−E1ℏ\omega = \dfrac{E_2 - E_1}{\hbar} ω=ℏE2​−E1​​

这正是量子力学版的拍频（beat frequency）——两个能级之差除以 ℏ\hbarℏ。以后会看到，原子的辐射频率、量子跃迁的"节奏"，本质上都来自这种叠加态的时间振荡，而频率永远对应着两个能级之差（ΔE=ℏω\Delta E = \hbar\omegaΔE=ℏω，呼应着普朗克—爱因斯坦关系 E=hνE=h\nuE=hν）。

💡 一句话总结：定态是"静止的快照"，叠加态才是"流动的电影"；而电影播放的帧率，由能级差 ΔE/ℏ\Delta E/\hbarΔE/ℏ 决定。

八、结语：一个方程，统辖整个微观世界

让我们把薛定谔方程的几条要点收拢在一起：

  • 它是量子力学的一个基本假定，是最基本的方程。量子力学对粒子运动的研究，基本上归结为在各种条件下求薛定谔方程的解。
  • 它是线性偏微分方程，满足态叠加原理；它的解——波函数——一般是个复函数。
  • 它适用于非相对论粒子、且不发生实物粒子产生和湮灭的情形。
  • 它的正确性由实验检验：在各种具体情形下从方程得出的结论，与实验结果比较来验证。

一百多年的事实证明，对于描述原子和分子的绝大多数现象，甚至低能核物理的许多现象，薛定谔方程都极为成功。它是人类历史上最成功的物理方程之一。

回过头看，量子力学解决问题的"标准流程"如今已经完整地铺在我们面前：

写出体系的哈密顿算符 H^\hat{H}H^ → 解定态薛定谔方程 H^ψ=Eψ\hat{H}\psi=E\psiH^ψ=Eψ 得到能量和波函数 → 再由波函数求出其他一切物理量。

可真正动笔去解 H^ψ=Eψ\hat{H}\psi=E\psiH^ψ=Eψ，你立刻会发现：除了最简单的几种势场，这个方程极难求出解析解。所以物理学家发展出了一套"经典题型"——那些恰好能严格解出的势场，每一个都成了理解微观世界的里程碑：

  • 势垒 / 势阱：会引出隧穿效应（隧道贯穿）——粒子能"穿墙"；
  • 一维无限深方势阱：会看到能量量子化的最干净样本；
  • 线性谐振子：是分子振动、晶格振动的基石；
  • 氢原子：解开它的方程，电子的"轨道"、能级、光谱线，将一一从数学里"长"出来。

这些，就是接下来几篇的主角。从普朗克那个"无奈"的假设，到玻恩惊世骇俗的概率诠释，再到今天这条统辖微观世界的方程——量子的版图，正在一块块拼完整。

而这一切的"运动定律"，不过是薛定谔在 1926 年写下的那一行方程罢了。
