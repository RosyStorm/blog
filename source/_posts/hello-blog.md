欢迎来到**颜浩轩的博客**！本站基于 Hexo + Butterfly 主题搭建，记录学习与生活。

## 本站使用的技术

- 生成器：Hexo 6
- 主题：Butterfly
- 部署：GitHub Pages + GitHub Actions

## 代码示例

```bash
hexo new "我的新文章"
hexo server   # 本地预览
hexo g        # 生成静态文件
```
