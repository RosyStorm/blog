本文介绍**核工程与核技术**领域的基础知识，作为本专业学习笔记的开篇。

## 主要研究方向

- **反应堆物理**：中子输运理论、临界计算、反应性控制
- **核燃料循环**：铀浓缩、燃料元件制造、乏燃料后处理
- **辐射防护与剂量学**：屏蔽设计、辐射监测、生物效应
- **核安全**：事故分析、严重事故管理、概率安全评价（PSA）

## 常用工具

```bash
# 蒙特卡洛中子输运计算（MCNP/OpenMC 等）典型流程示例
openmc -s 8          # 8 进程并行
openmc_plot_tally.h5 # 查看计数结果
```

## 典型代码

```python
# 简单的放射性衰变计算示例
import math
N0 = 1000          # 初始原子数
half_life = 5730   # 半衰期（年，碳-14）
t = 10000          # 经过时间
N = N0 * math.exp(-0.693 * t / half_life)
print(f"剩余原子数: {N:.1f}")
```
